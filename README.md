🎉系统技术架构：

1、.Net 7 + EF + REST + Vue + Element UI；

2、前后端分离；

3、支持多数据库，比如：MSSQL、MySQL。

前端地址：https://github.com/To-Effort/vue-element-admin

前端框架
vue-element-admin  
地址：https://github.com/PanJiaChen/vue-element-admin

前提：
node.js版本：=>16.18.0



功能：
用户管理，角色权限管理，菜单管理，日志管理
